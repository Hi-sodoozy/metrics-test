Metrics View User Guide

Quick start

  1. Open the Input page and set your metric title, goal, and current value.
  2. Select “Add to Display” to show a metric on the Display.
  3. Press Save to keep your changes and remain on the same screen, or press View Display to save and open the Display.

How to use key features

  • Summary view: Shows only Goal + Current from the summary fields.
  • Individuals view: Uses the Individual data table for rankings and progress.
  • Invites: Add team members as Viewer, Editor, or Admin per metric board.
  • Display mode: Use Light/Dark mode for office screens.

FAQ

Why is my metric not on the display?
Make sure “Add to Display” is turned on and saved.

Why are individual rows not showing?
Set “Display as” to Individuals and ensure row values are filled in.

Can I control who edits data?
Yes. Use invite permissions: Viewer, Editor, and Admin.

Will the display refresh after save?
Yes. Saving sends update signals, and the Display also refreshes periodically.

Troubleshooting

  • Changes not appearing: Press Save again and refresh your browser tab.
  • Wrong board open: Check if you are on your own board or an invited board.
  • Invite issues: Confirm email address and permission level, then resend invite.
  • Still stuck: Log out, log back in, and try again.
